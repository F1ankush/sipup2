## â“ FAQ

**Q: Where do messages go?**
A: To the database `contact_messages` table. Admins view them in the admin panel.

**Q: Can I delete messages?**
A: Yes, in the admin Messages panel, click Delete for any message.

**Q: Can users see replies?**
A: Currently, admins can see replies in the panel. You can add email notifications later.

**Q: Is it secure?**
A: Yes! Uses CSRF tokens, input validation, and sanitization.

**Q: Can I customize the form?**
A: Yes! Edit `/pages/contact.php` to add/remove fields.
