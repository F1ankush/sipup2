TECHNICAL DETAILS:
==================

CSS Structure:

/* Desktop - Hide mobile button items */
.navbar-button-item {
    display: none;
}

/* Desktop - Show navbar-buttons */
.navbar-buttons {
    display: flex;
    gap: 1rem;
    align-items: center;
}

/* Mobile - Show mobile button items */
@media (max-width: 768px) {
    .navbar-button-item {
        display: list-item;
        width: 100%;
    }
    
    /* Mobile - Hide desktop buttons section */
    .navbar-buttons {
        display: none;
    }
}

HTML Pattern:

<!-- Buttons on desktop (hidden on mobile) -->
<div class="navbar-buttons">
    <a href="pages/apply.php" class="btn btn-primary">Apply</a>
    <a href="pages/login.php" class="btn btn-secondary">Login</a>
</div>

<!-- Same buttons in menu (hidden on desktop) -->
<ul class="navbar-menu">
    <li><a href="index.php">Home</a></li>
    <li><a href="pages/about.php">About</a></li>
    ...
    <li class="navbar-button-item">
        <a href="pages/apply.php" class="btn btn-primary">Apply</a>
    </li>
    <li class="navbar-button-item">
        <a href="pages/login.php" class="btn btn-secondary">Login</a>
    </li>
</ul>
