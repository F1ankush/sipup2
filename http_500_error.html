üîç What Happened?

The server encountered an error while trying to process your request. This is typically a database connection issue.

Most Common Causes:

  • MySQL server is not running
  • Database credentials are incorrect
  • Database 'b2b_billing_system' doesn't exist
  • MySQLi PHP extension is not enabled

üöÄ Quick Fix Guide

Step 1: Start MySQL Server

For XAMPP Users:

1. Open XAMPP Control Panel

2. Find "MySQL" and click "Start" button

3. Wait for it to show "Running" with green indicator

4. Refresh this page

Step 2: Verify Database Exists

Check via phpMyAdmin:

1. Open phpMyAdmin: http://localhost/phpmyadmin

2. Look for 'b2b_billing_system' in the database list on left

3. If not present, click "New" and create it with UTF-8 charset

Step 3: Check Configuration

File: includes/config.php

Verify these settings for XAMPP:

define('DB_HOST', 'localhost'); define('DB_USER', 'root'); define('DB_PASS', ''); // Empty for XAMPP default define('DB_NAME', 'b2b_billing_system');
