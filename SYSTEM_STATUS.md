## Quick Start in 3 Steps

### Step 1: Configure (2 minutes)
```php
Edit: C:\xampp\htdocs\top1\includes\config.php

Update:
- DB_HOST = 'localhost'
- DB_USER = 'root'
- DB_PASS = 'your_password'
- DB_NAME = 'b2b_billing_system'
- ADMIN_SETUP_KEY = 'secure_key'
```

### Step 2: Setup Database (2 minutes)
```
1. Open phpMyAdmin
2. Create database: b2b_billing_system
3. Import: database_schema.sql
4. Done!
```

### Step 3: Create Admin (1 minute)
```
1. Visit: http://localhost/top1/admin/setup.php
2. Enter setup key
3. Create admin account
4. Login!
```
