QUICK 5-MINUTE SETUP

1. START SERVICES

   - Update database credentials (usually just database name)

   - User: root
   - Password: (leave blank for XAMPP default)

   - Open: http://localhost/top1/admin/setup.php

   - Click "Create Admin Account"

5. LOGIN AS ADMIN
   - Open: http://localhost/top1/admin/login.php
   - Username: admin
   - Password: Admin@123456
   - Click "Login"

6. ADD SAMPLE PRODUCTS
   - Click "Products" menu
   - Click "+ Add New Product"
   - Add 3-5 sample products with prices and stock quantities

7. TEST RETAILER SIGNUP
   - Open: http://localhost/top1/pages/apply.php
   - Fill form with test data
   - Submit application
   - Go back to admin panel
   - Click "Applications"
   - Approve the application
   - Create password for retailer

8. TEST RETAILER LOGIN

9. VERIFY PAYMENT (ADMIN)
   - Go back to admin
   - Click "Payments"
   - Approve the pending payment

10. GENERATE BILL (ADMIN)
    - Click "Orders"
    - Generate bill for the completed order
    - Download and verify

  4. Click "Download"

================================================================================
TROUBLESHOOTING
================================================================================

BLANK PAGE OR ERROR?
  1. Check C:\xampp\htdocs\top1\ exists
  2. Start Apache and MySQL
  3. Check error log in C:\xampp\logs\
  4. Verify database is created
  5. Check config.php database settings

CAN'T LOGIN?
  1. Check if MySQL is running
  2. Verify admin account was created
  3. Check caps lock on password
  4. Browser cookies enabled?
  5. Clear browser cache

UPLOAD NOT WORKING?
  1. Check uploads/ folder exists
  2. Verify permissions are correct
  3. Check file size limit (5MB default)
  4. Allowed formats: JPG, PNG only
  5. Check disk space

DATABASE ERROR?
  1. Open phpMyAdmin: http://localhost/phpmyadmin/
  2. Check if b2b_billing_system database exists
  3. Check if tables are created
  4. Re-import database_schema.sql if needed
